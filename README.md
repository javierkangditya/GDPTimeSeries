# GDP Time Series Analysis (1980-2028)

## Author
Javier Kangditya

## Project Overview
This project performs a comprehensive analysis of global GDP per capita from 1980 to 2023, with forecasts extending to 2028. The goal is to explore historical economic trends and predict future GDP trajectories for selected countries, focusing on ASEAN-5.

---

## Objectives
- Explore GDP trends and patterns across countries.
- Engineer time series features such as lagged GDP values, rolling averages, and growth rates.
- Forecast GDP using **ARIMA** and **ETS** models.
- Evaluate model performance using **MAE**, **RMSE**, and forecast visualizations.
- Provide insights and actionable recommendations for economic planning.

---

## Dataset
**IMF's GDP Data: 1980-2028 Global Trends**  

**Columns:**
- `Country` : Name of country or region  
- `1980-2023` : Actual GDP per capita (USD)  
- `2024-2028` : Projected GDP per capita (USD)  

**Source:** [IMF World Economic Outlook](https://www.imf.org/en/Data)
